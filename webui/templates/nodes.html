🖥️ Управление нодами

Ноды — это серверы, на которых работают ваши MTProto прокси. Вы можете добавить несколько серверов в разных локациях для повышения надёжности и скорости. Каждая нода запускает MTG Agent — лёгкий сервис для мониторинга и управления.

➕ Добавить ноду

Добавьте удалённый сервер. На нём нужно установить MTG Agent (инструкция ниже). После добавления вы сможете проверять связь, синхронизировать клиентов и управлять прокси.

📦 Установка агента на удалённый сервер

Для добавления ноды нужно установить MTG Agent на удалённый сервер. Выполните эту команду на удалённом сервере, заменив YOUR_TOKEN на ваш секретный токен:

curl -fsSL https://raw.githubusercontent.com/Therealwh/MTProtoSERVER/main/agent/install.sh | bash -s -- --port 9876 --token YOUR_TOKEN

После установки добавьте ноду в форму выше, указав IP, порт (9876) и тот же токен.
