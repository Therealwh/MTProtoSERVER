📱 Как использовать SOCKS5

Инструкции по подключению к SOCKS5 прокси из различных приложений.

🌐 Браузер: Настройки → Прокси → SOCKS5 → IP: {{ settings.proxy_ip }}, порт: {{ settings.socks5_port }}, логин: {{ settings.socks5_user }}
📱 Telegram: Настройки → Данные и память → Прокси → SOCKS5 → введите данные
💻 Системный прокси: Настройте SOCKS5 в настройках сети вашей операционной системы
🔧 cURL: curl -x socks5://USER:PASS@{{ settings.proxy_ip }}:{{ settings.socks5_port }} https://ifconfig.me
